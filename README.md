# 宇宙可视化系列 · A Computed Universe

十五部交互式天体物理可视化，从大爆炸到黑洞蒸发，从夸克到宇宙之网。

**没有一帧是预先做好的动画** —— 每一页都在浏览器里实时求解真实的物理方程（测地线 / N 体 / ODE / 波函数），
配有独立 Node 数值内核对拍与无头浏览器验证闸门；能算的一定真算，不能算的在各页「声明」里逐条列明。

在线访问：https://zhaopuyuan001-lgtm.github.io/cosmos-visualizations/
